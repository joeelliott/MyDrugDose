# Joe's Neonatal & Pediatric Dosing + Infusion Calculator (CLI)

A lightweight command-line medication calculator for neonatal and pediatric patients.  
Supports weight-based dosing, infusion rate calculations, and fast drug lookup.

**For education and reference only.  
Always verify doses with current institutional protocols, on-call physicians, and pharmacists.**

---

## Features

### ✔ Weight-Based Single-Dose Calculations
- Enter patient weight (kg)
- Select a drug
- Calculator applies per-kg dosing and maximum dose limits
- Outputs fully explained math

### ✔ Infusion Rate Calculator (mL/hr)
Handles drugs using:
- `mcg/kg/min`
- `mcg/kg/hr`
- `unit/kg/hr`

You provide:
1. Desired infusion dose  
2. Bag/syringe concentration (e.g., “4 mg in 50 mL”)  
3. Calculator outputs **mL/hr**, including transparent steps  

### ✔ Searchable Drug Database
Search by any part of a drug name (e.g., `epi`, `d10`, `dop`, `ket`).

### ✔ Neonatal + Pediatric Support
Filter drugs by:
- Pediatric
- Neonatal
- All
